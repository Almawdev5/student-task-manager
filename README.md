# 🧑‍🎓 Student Task Manager (CLI)

A simple command-line based task management system built with Python.  
This project demonstrates Git workflow including branching, merging, and conflict resolution.

---
## 🛠 Tech Stack

- **Language:** Python
- **Version Control:** Git
- **Hosting:** GitHub
- **Storage:** JSON file (local persistence)
- **Interface:** Command Line Interface (CLI)

## 🚀 Features

- ➕ Add tasks
- 👀 View all tasks
- ✅ Mark tasks as completed
- ❌ Delete tasks
- 💾 Persistent storage using a JSON file
